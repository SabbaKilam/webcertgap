In Depth

  • What?
    • It is proposed that a Web Development Certificate be offered at PIT.

  • Why?
    • This will offer an attractive, short-term path to a solid career as a Web Developer after just two semesters. It is thought that offering an intensely-focused technical certificate after just eight months of study will attract a broader range of students to computer science.

    The option of continuing school for another two semesters would still be open to those seeking an associates degree, but students finish their first year undistracted by less relevant courses, with a solid set of skills, and a few industry-recognized certifications, to pursue their career immediately, if they choose to do so.

  • When?
    • The Web Development Certificate program should begin September, 2015.

Course Content

Web Development is not just Web Design. Web developers create dynamic interactive websites using the best mix of design principles and programming. Students will first master the "Three Amigos" of the client-side web development, namely
  • HTML - for structure, layout and content
  • CSS - to style and beautify
  • JavaScript - the language for dynamic interactive web pages

............. etc.


.... and finally students learn basic Website Management, which includes:
  • Monetizing your site
  • Protecting it against Security Threats
  • optimizing its content and performance for search engines
