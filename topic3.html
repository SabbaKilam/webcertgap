Computer Science Curriculum

Web Development Certificate


This program will focus on the growing field of web programming, also known as web development. Students can take the programming portion of the program either before or after the general education requirements.

Web Programming Courses

(all are 4 credit classes except where noted)
  1. HTML5 & CSS3
  2. Client Programming: JavaScript & JQuery
  3. Photoshop and Dreamweaver
  4. Server Programming I: Ruby & PHP
  5. Server Programming II Node.JS
  6. Unix (3 credits)
  7. SQL & noSQL (3 credits)
  8. Mobile Programming
  9. Website Management

General Education Requirements

(all are 3 credit classes except where noted)
  1. Composition 1
  2. Composition 2
  3. Discrete Math I
  4. Discrete Math II
  5. Calculus (4 credits)
  6. Critical Thinking in the Modern Age
  7. Communications and Social Interaction
  8. Humanities Elective

Web Programming Certificate = 38 credits
General Education = 25 credits
Total for A.S. Degree = 63 credits

Much like the MA courses taught in SPP, we suggest that the Web Programming courses be run in a cohort style to allow for at least a 2nd entry point for students.
